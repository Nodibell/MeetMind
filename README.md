# MeetMind 🧠🎤

**MeetMind** — це сучасний macOS додаток для автоматичної транскрипції нарад та генерації резюме за допомогою штучного інтелекту. Весь процес відбувається локально на вашому пристрої, забезпечуючи максимальну приватність.

![App Icon](meetmind_app_icon_final_1778091152707.png)

## Основні можливості ✨

- **Локальна транскрипція**: Використовує Apple WhisperKit для високоякісного розпізнавання мови прямо на вашому Mac (CPU/Neural Engine).
- **Підтримка системного аудіо**: Записуйте мікрофон, системне аудіо (Zoom/Google Meet) або обидва одночасно.
- **AI Резюме через Ollama**: Автоматичне створення ключових тез, завдань та рішень за допомогою локальних LLM (Llama 3, Mistral тощо).
- **Експорт в Obsidian**: Пряма інтеграція з вашим Obsidian vault через Security-Scoped Bookmarks.
- **Privacy First**: Ваші дані ніколи не залишають ваш пристрій. Жодних хмарних API для транскрипції.
- **Сучасний дизайн**: Sleek macOS інтерфейс із підтримкою Dark Mode та Glassmorphism.

## Технологічний стек 🛠

- **Swift & SwiftUI**: Сучасний інтерфейс та висока продуктивність.
- **WhisperKit**: Порт OpenAI Whisper від Argmax для CoreML.
- **Ollama**: Локальний запуск LLM.
- **SwiftData**: Надійне збереження історії нарад.
- **ScreenCaptureKit**: Високоякісне захоплення аудіо.

## Як почати 🚀

### Вимоги
- macOS 14.0+
- [Ollama](https://ollama.com/) (встановлена та запущена)
- Моделі Whisper (завантажуються автоматично при першому запуску)

### Встановлення
1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/yourusername/MeetMind.git
   ```
2. Відкрийте `MeetMind.xcodeproj` в Xcode.
3. Переконайтеся, що всі Swift Packages завантажені.
4. Натисніть **Run** (Cmd + R).

## Налаштування ⚙️

1. **Obsidian**: Оберіть папку вашого vault у налаштуваннях додатка для автоматичного експорту.
2. **Ollama**: Оберіть модель (наприклад, `llama3`) та вкажіть ендпоінт (за замовчуванням `http://localhost:11434`).
3. **Мова**: Оберіть основну мову транскрипції (Українська/English) або залиште авто-визначення.

## Приватність 🔒

MeetMind не збирає жодних даних. Транскрипція виконується локально через CoreML, а резюме генерується локальним екземпляром Ollama.
